# 读书行动派 MVP

没时间读书，但想要书里的知识 → AI 帮你把一本书变成「跟你有关的启发 + 能立刻做的事」。

## 核心设计

- **输入零摩擦**：输入书名，或直接拍一张封面照片（多模态模型识别书名+作者），不需要下载/上传电子版全文
- **不做整书复述**：只做基于书名的转化式二次创作（类似书评人的讲书），规避版权风险
- **个性化，而非通用摘要**：按你的角色 + 当下具体问题，生成专属的启发和行动清单
- **缓存网络效应**：同一本书的"基础解读"做缓存，第二个读者复用基础解读，只重新生成个性化层，边际成本随用户量增长下降

## 快速开始

```bash
cd book_action_mvp
pip install -r requirements.txt
copy .env.example .env
# 编辑 .env，填入 LLM_API_KEY（兼容 OpenAI 协议：DeepSeek/通义/腾讯混元等）
python app.py
```

打开 http://127.0.0.1:5800

> 未配置 `.env` 时会自动进入**演示模式**，UI 全流程可点击体验，但内容是占位文案。

## 目录结构

```
book_action_mvp/
  app.py              # Flask 后端：书籍识别 + 基础解读缓存 + 个性化生成
  static/index.html   # 前端单页（手机端优先，拍封面/输入书名 → 生成知识卡片）
  cache/book_cache.json  # 基础解读缓存（运行后自动生成）
  requirements.txt
  .env.example
```
